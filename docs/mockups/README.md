# Dashboard mockups

Static HTML/CSS explorations for the cases-dashboard toolbar redesign.
These are reference artefacts only — not served by the app.

To view: open `index.html` directly in a browser
(`file://.../docs/mockups/index.html`), or run a one-off local server in
this folder (`npx serve .` or `python -m http.server 8080`).

The mockups use the Tailwind CDN (`cdn.tailwindcss.com`) and the Lucide
UMD bundle from unpkg. They're for design discussion, not production —
do not link to them from app code.

| # | File | Concept |
|---|---|---|
| 1 | `01-condensed.html` | Single-row sticky toolbar (Linear/Notion) |
| 2 | `02-hero-stats.html` | Greeting + 4 KPI cards + 2-row toolbar |
| 3 | `03-command-bar.html` | Ctrl+K command bar, filters as a readable sentence |
| 4 | `04-inverted.html` | Table-first; toolbar lives inside the table card |
